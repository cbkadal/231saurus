# Ch03: Cheat Sheet
by BinKadal, Sdn. Bhd.

## Git: Reset

```
git reset --hard HEAD
git clean -df

```

## Init Docusaurus

```
npm init docusaurus@latest Docusaurus classic

```

## Start Docusaurus

```
npm run start -- --host 0.0.0.0

```

## Deployment To GitHub

```
yarn deploy

```

## Fix NPM

```
npm audit fix

```
